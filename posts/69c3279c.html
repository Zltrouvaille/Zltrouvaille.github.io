Git

设置用户签名和邮箱

git config –global user.name

git config –global user.email

初始化本地库

git init

查看本地库的状态

git status

  1. 红色的知识存在于工作区
  2. 绿色是存在暂存区
  3. 没有东西可以提交

添加暂存区

git add 文件名

提交本地库

git commit -m’’日志信息’’（版本）文件名

查看信息

git relog

详细 git log能够知道谁提交的

版本穿梭

git reset –hard 版本号

分支

副本指针的引用

分支的操作

查看分支: git branch -v

创建分支:git branch 分支名

切换分支:git checkout 分支名

合并分支：git merge 分支名

​ 注意：

  • 提交 时候不能加上文件名子

GItHub

别名

git remote -v （查看当前所有的远程地址的别名）

git remote add 别名 远程地址（设置别名）

推送(分支)

git push git-demo master

拉取

git pull git-demo master

克隆

克隆：git clone 地址

团队协作

加入团队后才能相互推送

shh免密登录

ssh-keygen -t rsa -C 3027638938@qq.com
