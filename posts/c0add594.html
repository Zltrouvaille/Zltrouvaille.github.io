HTML

day1

  1. border-radius: 圆角(px越大越圆)
  2. postition: absolute –绝对位置
  3. input type radio: 单选(给他们去相同的名字) checkbox :多选
    1. checked : 进入页面默认选择
    2. maxlength： 用户最多输入多少
    3. type:reset 重新输入所有
    4. type：butten: 普通真牛牛
    5. type： file 上传文件用的
    6. text-indent：2em缩进两个字
  4. label: for 对应的input的name的值
  5. select：
    1. selected=”selected” 默认选择
  6. textarea ：反馈留言
    1. cols： 每行的字符数
    2. row： 显示多少行

css

  1. font-family: 选择字体
  2. em，i： 倾斜 fontStyle：normal不倾斜
  3. text-align：center，本质在盒子内居中
  4. line-height 行高
  5. text-decoration 文本修饰 下划线啥的
  • 块元素：h,p,div,ul,ol.li

！文字类的标签不能在放块

  • 行内元素：<a,<span
    1. 可以多个
    2. 宽高无法设置
  • 行内块元素： <img,<input,td
    • 可以设置块的元素
    • 本身的行内元素，可以显示多个
  • ！
    • display: block 转块
    • display: lnline 转行
    • block-lnline 行内块元素
    • alt 选择c辅助

背景

  1. background-color：默认是透明的

盒子

  1. line-height：行高（有上空隙和下空隙）放在盒子里

emmet语法

  1. div*n
  2. ul>li(包含)
  3. +就是兄弟
  4. .是类，#是id
  5. p.one
  6. .demo$*5(类 自增)
  7. w100px
