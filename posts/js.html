HTML

day1

  1. border-radius: 圆角(px越大越圆)
  2. postition: absolute –绝对位置
  3. input type radio: 单选(给他们去相同的名字) checkbox :多选
    1. checked : 进入页面默认选择
    2. maxlength： 用户最多输入多少
    3. type:reset 重新输入所有
    4. type：butten: 普通真牛牛
    5. type： file 上传文件用的
    6. text-indent：2em缩进两个字
  4. label: for 对应的input的name的值
  5. select：
    1. selected=”selected” 默认选择
  6. textarea ：反馈留言
    1. cols： 每行的字符数
    2. row： 显示多少行

css

  1. font-family: 选择字体
  2. em，i： 倾斜 fontStyle：normal不倾斜
  3. text-align：center，本质在盒子内居中
  4. line-height 行高
  5. text-decoration 文本修饰 下划线啥的
  6. font-size: 14px/1.5 1.5是字体的1.5倍
  7. body是12px
  • 块元素：h,p,div,ul,ol.li

！文字类的标签不能在放块

  • 行内元素：<a,<span
    1. 可以多个
    2. 宽高无法设置
  • 行内块元素： <img,<input,td
    • 可以设置块的元素
    • 本身的行内元素，可以显示多个
  • ！
    • display: block 转块
    • display: lnline 转行
    • block-lnline 行内块元素
    • alt 选择c辅助
  • 权重

继承<元素<类<id<行内<!important

无论父元素是多少子是0

a< 有自己的a{color : blue}

margin: 0 auto;左右水品剧中

背景

  1. background-color：默认是透明的

盒子

  1. line-height：行高（有上空隙和下空隙）放在盒子里
  • border：边框
    • solid：实现边框（border.width:n px,border.style: soild:)5px soild red
    • 单元格table,td,th
      • border-collapse : 合并他的边框
      • text-align: center; 文字居中
  • panding : 与内容的距离
    • 1：上下左右
    • 2：上下一个，左右一个
    • 3：上 左右 下
    • 4 上 右 下 左 顺时针
    • 固定大小的盒子，加上 内边距会影响 盒子的大小
  • margin:

父子 子有外边距 会塌陷

  • 解决办法：
    • transparent 透明
    • 给一个内边距：padding
    • orverflow：hidden
  • 清除 边距
      • margin:0
      • padding:0
  • 给行内元素加上浮动就有了行内块特点
  • 只会影响前面的标准流
  • 清除浮动
    • 块级元素：clear：both
    • 给父级添加 overflow : hidden
  • 定位：position
    • static：静态定位
    • relative：相对定位
      • 再到自己原来的位置移动
    • absolut： 绝对定位
      • 没有祖先，祖先没定义
      • 最近的祖辈 根据他的位置 以他为基准点
      • 设有绝对定位，不占有原来的位置
      • 子绝父相
      • em 是行内标签，可以使用 图片的叠加 在同一个li中两个img
    • fixed： 固定定位
      • 参照浏览器的位置
      • 不占有原先的位置
      • 小技巧
          1. left50%
          2. margin-lift版新的一半
    • sticky
      • 必须要添加 top或者left right bottom
    • z-index
      • 如果没有写就是aoto谁在前面谁就
      • 只有定位才有
      • 数字前面不能加单位

​ 加了绝对定位 不能田间margin：0 auto水平居中

  • 让他水平居中
    • left ： 50%
    • left ： -n/2
  • 加了决定对或者固定定位
    • 可以直接设置高度和宽度
    • 块级元素不给高宽 加了定位的话 默认就是内容的大小
  • 浮动和决定
    • 浮动不压，决对压住

emmet语法 -

  1. div*n
  2. ul>li(包含)
  3. +就是兄弟
  4. .是类，#是id
  5. p.one
  6. .demo$*5(类 自增)
  7. w100px

显示与隐藏

  • display:
    • 显示：block
    • 不显示none
    • 不显示不占有有；
  • visibility
    • visible ： 可见
    • hidden： 隐藏
  • overflow：
    • 默认显示
    • hidden： 多出来的文字隐藏
    • scroll：溢出的地方显示滚动条
    • aoto：在需要得放提案加滚动条
    • 有超出的的部分最好别用：hidden
  • link : 初始化的状态
  • visited: 被访问过的状态
  • active: 正在访问的状态
  • hover: 鼠标悬浮的状态
